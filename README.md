## Installation

MongoDB Compass [MONGODB](https://www.mongodb.com/try/download/community)

Install the dependencies and devDependencies and start the server.

```sh
$ brew install mongodb-atlas
$ atlas setup
```

For production environments...

```sh
npm install nodemon
npx nodemon app.js

```
